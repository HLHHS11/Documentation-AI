## 環境構築
``` sh
# リポジトリのクローン
git clone https://github.com/HLHHS11/Documentation-AI.git
# Python仮想環境の構築
python3 -m venv venv
# 仮想環境をアクティベート
source venv/bin/activate # Windowsの場合venv\Scripts\activate
pip install -r requirements.txt
```

## `OPENAI_API_KEY`のソース定義ファイル

以下は`OPENAI_API_KEY`のソース定義ファイルの内容です。

```python
OPENAI_API_KEY = os.environ.get('OPENAI_API_KEY')
```

このファイルは、`os.environ.get`関数を使用して環境変数`OPENAI_API_KEY`の値を取得し、`OPENAI_API_KEY`変数に代入しています。
